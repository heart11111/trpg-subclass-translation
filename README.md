# 화살성채 캠페인 자료

GitHub Pages로 공유하는 화살성채 캠페인 자료 모음입니다.

- `index.html`: 자료 허브
- `player-guide.html`: 플레이어 가이드 (1부 왈든)
- `crossroads-guide.html`: 크로스로드 플레이어 가이드 (2부 · Lv.5 시작)
- `world.html`: 세계관 자료
- `setting.html`: 설정 정리
- `homebrew.html`: 홈브류 서브클래스 정리
- `source/`: 원본 Markdown 자료
- `docs/ima2-gen-guide.md`: ima2-gen(로컬 이미지 생성 도구)로 카드 아트를 만들어 이 저장소에 반영하는 절차
